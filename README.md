## Overview
This project is an end-to-end sentiment analysis application built with Python and Streamlit.
It allows users to analyze text sentiment in real time using NLP techniques and visual insights.

## Business Use Case
- Understand customer sentiment from reviews or feedback
- Support marketing and brand perception analysis
- Enable fast qualitative insight without complex tools

## Tech Stack
- Python (Pandas, NumPy)
- NLP (Text preprocessing, sentiment scoring)
- Streamlit (UI & deployment)
- Matplotlib / Seaborn (visualization)

## Key Features
- Real-time sentiment classification
- Clean, interactive UI
- Deployed live for stakeholder access

## What I Learned
- Translating NLP models into business-ready tools
- Deploying analytics apps for non-technical users
- Communicating insights visually
